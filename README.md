# Pipelined MIPS SoC with GPIO

## About
This project implements a 5-stage pipelined MIPS CPU with hazard detection and data forwarding, integrates instruction/data memory, and extends the system with a memory-mapped GPIO peripheral. It includes testbenches, simulation artifacts, and detailed design documentation for the pipeline transformation and SoC integration.
